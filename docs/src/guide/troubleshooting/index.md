## Getting Help

If you're still experiencing issues, [check existing
issues](https://github.com/jolars/tomat/issues) on GitHub and the [discussion
board](https://github.com/jolars/tomat/discussions) for similar problems and
solutions.

If you cannot find a solution, please open a new issue or discussion.
